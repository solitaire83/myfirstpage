Ce este HTML-ul?

HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afișate într-un browser (sau navigator). Scopul HTML este mai degrabă prezentarea informațiilor paragrafe, fonturi, tabele ș.a.m.d. decât descrierea semanticii documentului. În cadrul dezvoltării web de tip front-end, HTML este utilizat împreună cu CSS și JavaScript.

Taguri si elemente populare in HTML

Detalii despre tagul <P>:

  • Unul dintre cele mai folosite taguri.
  • Este folosit pentru paragafe

Un alt element popular este Headingul. Headingurile pot fi de mai multe tipuri. Ele se diferentiaza prin numere.

  3. <H3>

  6. <H6>

Spanul este un subelement>, folosit pentru a selecta si manipula un cuvant o propozitie sau o fraza.

br (break line) este un selfclosing tag folositor pentru a adauga spatiu in texte.
Un alt selfclosing tag similar este <hr> (horizontal rule) care adauga un spatiu orizontal ce separa continutul din pagina HTML.

Body-ul, <body> reprezinta aspectul documentului HTML (vizual).

Strucutra paginii HTML

Head, <head> reprezinta un container de date ce nu sunt afisate vizual.

<title>, folosit pentru a seta un titlu paginii HTML.

<link> defineste relatia dintre pagina html si o resursa externa.
De cele mai multe ori acest tag este folosit pentru a stiliza pagina html folosind un fisier css extern sau pentru a adauga un favicon paginii.

Inputul este un selfclosing tag, flosit pentru a introduce date.
Inputurile pot fi de mai multe tipuri:

  1. text (default)
  2. number
  3. email
  4. password
  5. checkbox
  6. radio
  7. submit (creeaza un buton de submit)
  8. ...

Inputurile mai sunt frecvent insotite de labeluri <label> care sunt corelate prin atributele id si for.

<img> este folosit pentru a incorpora o imagine in pagina HTML.
<video> este folosit pentru a incorpora un video in pagina HTML.
<audio> este folosit pentru a incorpora un continut audio in pagina HTML.

Tagul <button> incorporeaza un buton.

Tagul <a> este folosit pentru a incorpora link-uri in pagina html.
Cel mai important atribut pentru <a> este href care indica destinatia link-ului.
By default, link-urile vor aparea:

  • Un link nevizitat o sa apara subliniat si albastru.
  • Un link vizitat o sa apara subliniat si mov.
  • Un link activ o sa apara subliniat si rosu.

Tagul <div> este folosit ca si un container pentru elementele HTML si poate fi stilizat/manipulat folosind CSS si Javascript.
Un alt tag similar este <section> care este mai bine indexat de catre motoarele de cautare.
