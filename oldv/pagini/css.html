Ce este CSS-ul?

CSS sau Cascading Style Sheets este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atașa elementelor HTML prin intermediul unor fișiere externe sau în cadrul documentului, prin elementul și/sau atributul style.
CSS se poate utiliza și pentru formatarea elementelor XHTML, XML și SVGL. CSS este una dintre tehnologiile de bază utilizate în procesul de dezvoltare web, împreună cu HTML și JavaScript.

Proprietati si valori populare in CSS

Proprietati pentru font/text:

  • font-family
  • Exemplu: font-family: Arial, Helvetica, sans-serif;
  • Descriere: font-family este de fapt o lista de fonturi din care browserul va folosi in ordinea in care le recunoaste (primul folosit va fi primul din lista, daca nu este recunoscut il foloseste pe al doilea si tot asa mai departe).
  • font-weight
  • Exemplu: font-weight: 400;
  • Descriere: Proprietatea font-weight specifica greutatea fontului folosind valori numerice sau BOLD, BOLDER, LIGHTER.
  • font-size
  • Exemplu: font-size: 30px;
  • Descriere: Prin aceasta proprietate stabilim dimensiunea fontului care poate fi exprimata in pixeli (px), puncte (pt) sau procente (%)
  • text-decoration
  • Exemplu: text-decoration: none;
  • Descriere: Proprietatea text-decoration este utilizată pentru a seta sau elimina decorațiunile din text.
  • text-align
  • Exemplu: text-align: center;
  • Descriere: Prin aceasta proprietate putem pozitiona textul la dreapta, stanga sau centru.

Proprietati pentru margini/dimensiuni/bordere:

  • width
  • Exemplu: width: 500px;
  • Descriere: Folosind proprietatea width stabilim latimea elementelor.
  • height
  • Exemplu: height: 500px;
  • Descriere: Folosind proprietatea height stabilim inaltimea elementelor.
  • margin
  • Exemplu: margin: 15px;
  • Descriere: Aceasta proprietate se foloseste pentru a creea spatiu intre elemente sau inafara bordurilor. Se pot adauga margini in directiile (sus, jos, stanga si dreapta).
    Pentru a stabili in ce directie se adauga spatiu, se folosesc urmatoarele proprietati:
    margin-top
    margin-bottom
    margin-left
    margin-right
  • padding
  • Exemplu: padding: 15px;
  • Descriere: Aceasta proprietate se foloseste pentru a creea spatiu intre continutul elementelor si marginile acestora. Se pot adauga padding-uri in directiile (sus, jos, stanga si dreapta).
    Pentru a stabili in ce directie se adauga spatiu, se folosesc urmatoarele proprietati:
    padding-top
    padding-bottom
    padding-left
    padding-right
  • border
  • Exemplu: border: 5px solid;
  • Descriere: Aceasta proprietate se foloseste pentru a adauga borduri in jurul elementelor. Se pot adauga borduri in directiile (sus, jos, stanga si dreapta).
    Pentru a stabili in ce directie se adauga bordurile, se folosesc urmatoarele proprietati:
    border-top
    border-bottom
    border-left
    border-right

Proprietati pentru pozitie/afisaj:

  • position
  • Exemplu: position: absolute;
  • Descriere: Aceasta proprietate se foloseste pentru a stabili modul in care un element este pozitionat in cadrul paginii. Exista mai multe tipuri de pozitionare care determina comportamentul elementului.
    Valorile de pozitionare sunt urmatoarele:
    static
    relative
    absolute
    fixed
    sticky
  • display
  • Exemplu: display: flex;
  • Descriere: Aceasta proprietate se foloseste pentru a stabili modul in care un element este afisat in pagina. Proprietatea display controleaza atat comportamentul vizual, cat si comportamentul structural al unui element.
    Valorile sale sunt urmatoarele:
    block
    inline
    inline-block
    flex
    grid
