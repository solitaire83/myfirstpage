HTML <img> Tag

<img>
- Acest element HTML este folosit pentru a incorpora imagini in pagina.
- Incoroporarea unei imagini se poate face atat extern(link-ul web al unei imagini) cat si intern (locatia imaginii din pc) prin intermediul atributului src (sursa).
Atribute frecvent folosite:
- alt (Folosit ca alternativa pentru o imagine nefunctionala si se afiseaza ca si text)
- width (Folosit pentru a seta latimea imaginii)
- height (Folosit pentru a seta inaltimea imaginii)

Exemplu:
<img src="https://link-imagine.ro/imagine.png" alt="Text Alternativ" width="1920" height="1080"> (extern)
<img src="../imgs/imagine.png" alt="O imagine" width="220" height="220"> (intern/local)





HTML <video> Tag

<video>
- Acest element HTML este folosit pentru a incorpora videouri in pagina.
- Incorporarea unui video se poate face atat intern cat si extern prin intermediul tagului <source> cu al sau atribut src.
Atribute frecvent folosite in <video>:
- controls (Face videoul sa fie controlabil)
- width (Folosit pentru a seta latimea videoului)
- height (Folosit pentru a seta inaltimea videoului)

Exemplu:
<video width="220px" height="220px" controls> <source src="locatie_video.mp4" type="video/mp4"> </video>

HTML <audio> Tag

<audio>
- Acest element HTML este folosit pentru a incorpora audiouri in pagina.
- Incorporarea unui audio se poate face atat intern cat si extern prin intermediul tagului <source> cu al sau atribut src.
Atribute frecvent folosite in <audio>:
- controls (Face audioul sa fie controlabil)
- autoplay (Audioul incepe redarea imediat dupa ce pagina este incarcata)
- loop (Sunetul o sa fie redat la nesfarsit dupa terminarea acestuia)

Exemplu:
<audio controls> <source src="locatie_audio.mp4" type="audio/mpeg"> </audio>
