Tabele

Tabelele HTML permit dezvoltatorilor web să aranjeze datele în rânduri și coloane.
Un tabel în HTML constă din celule de tabel în interiorul rândurilor și coloanelor.



Exemplu tabele:

Nr. Nume Prenume
1 Condruz Iulian
2 Condruz Iulian
Tag Descriere
<table> Tabela propriu-zisa
<tr> Table row/rand in tabela
<th> Table header/celula de antet în tabela
<td> Table data/celula de date în tabela
<thead> Table head/sectiunea de antet a tabelei
<tbody> Table body/corpul tabelei
Nume Prenume
Condruz Iulian
